# WME Scripts Collection

Hier findest du meine Userscripts für den **Waze Map Editor (WME)**.  

## Inhalt

- WME Userscripts (`*.user.js`) fuer verschiedene Aufgaben im Editor
- Scripts mit **WME JS SDK**

## Voraussetzungen

- Browser mit Violentmonkey, Greasemonkey oder Tampermonkey
  - getestet auf Tampermonkey und Violentmonkey, jedoch unterstützt Violentmonkey nicht viele WME Scripts
- Grundwissen in WME-Editing

## Installation

1. Tampermonkey installieren.
2. Gewuenschtes `*.user.js` Script oeffnen.
3. Script importieren und aktivieren.
4. WME Website neu laden, dann Script-Tab oder Panel im Editor pruefen.

## Script-Philosophie

- SDK-first statt fragiler DOM-Hacks
- klare, wartbare Struktur
- defensives Error-Handling
- UI kompakt und editorfreundlich
- Review-/QA-Fokus statt blindem Auto-Fix

## Hinweise

- Scripts arbeiten in der Regel nur auf **aktuell geladenen WME-Daten**.
- Je nach Region, Zoom und Datenstand kann das Verhalten unterschiedlich sein. Jedoch wurde es nahezu perfekt personalisiert.
- Immer gegenpruefen (Wazeopedia / lokale Mapping-Regeln).
